First-Time Setup
  1. In the Google Sheet, name the working tab ORDERS with the headers above. Add data-validation dropdowns on Status and WaitMin.
  2. Open Extensions → Apps Script, paste apps-script/Code.gs from the repo, and Save.
  3. Run the ensureV2 function once (adds columns Q–T incl. PickupAt + the LOG tab). Authorize when prompted.
  4. In Project Settings → Script Properties, add API_TOKEN = a long random string.
  5. Add an installable “On edit” trigger for onEditTrigger (backs up manual sheet edits).
  6. Deploy → New deployment → Web app: execute as Me (an editor account), access “Anyone with the link”. Copy the /exec URL. (After editing the script later, use Manage deployments → New version so the URL stays the same.)
  7. Share the sheet with op1 & op3 as Editors, op2 as Viewer, and turn OFF “anyone with the link”.
  8. Paste the /exec URL + token above, click Save & Test. Then set a Staff PIN (locks the Control & Check-in pages).
  9. Open the screens: Control on the warehouse iPad, Check-in at the sales window, and the TV pages on each display.
