Steps to Connect

  1. Open Google Sheets and create a new spreadsheet (or copy your existing one).
  2. Set up row 1 as headers: # · Customer Name · Invoice # · Boxes · Status · Paid · Notes
  3. Enter your orders starting from row 2. Set Status to one of: Not Pulled, Processing, Ready, Done.
  4. Go to File → Share → Publish to web. Under "Link", choose Comma-separated values (.csv), then click Publish.
  5. Copy the URL that appears. It will look like:
    https://docs.google.com/spreadsheets/d/…/pub?gid=0&single=true&output=csv
  6. Paste that URL into the field above and click Save & Test Connection.
  7. Open Warehouse and Customer Pickup links above in the TV browsers. Done!
✓ Daily Reset
At the end of each day, you can download the sheet as Excel, then clear all rows below the header for the next day — or simply keep adding rows. The TV screens always show whatever is currently in the sheet. No website changes needed.
